Weather API Project

About:

This is a simple weather API (Application Programming Interface) made using Python and Flask. The main idea behind this project was to let users check the current temperature of any city just by typing the city name in the URL. It connects to the OpenWeatherMap API, fetches live weather data, and shows the temperature in Celsius. It helped me understand how Flask works, how to use routes, and how to pull real data from the internet using APIs.

To view the project, please visit Weather-API (view on GitHub)

Features:

The website features the following key sections:

  • You can search the weather of any city by just typing it in the URL (like /weather/Delhi).
  • Shows the temperature in Celsius after converting it from Kelvin.
  • Uses the OpenWeatherMap API to get real-time weather data.
